Formatting in HTML

HTML allows you to format text and other elements to enhance the appearance of your web page. Here are some common formatting techniques:

Text Formatting

HTML provides various tags to format text, including:

  • <b> - Bold text
  • <i> - Italic text
  • <u> - Underlined text
  • <em> - Emphasized text
  • <strong> - Important text
  • <sup> - Superscript text
  • <sub> - Subscript text

Font Styles

You can also change the font style and size using CSS:

<style>
    p {
        font-family: Arial, sans-serif;
        font-size: 16px;
    }
</style>

Paragraph Formatting

HTML provides tags to format paragraphs, such as:

  • <p> - Defines a paragraph
  • <br> - Inserts a line break
  • <pre> - <p>This is a paragraph.</p> <p>Another paragraph.</p>
