HTML Tags

HTML tags are used to create elements on a web page. Each tag has a specific purpose and is used to define the structure and content of a web page. Here are some commonly used HTML tags:

<html>

The <html> tag represents the root of an HTML document.

<html>
    <!-- Document content -->
</html>

<head>

The <head> tag contains meta-information about the HTML document.

<head>
    <title>Page Title</title>
</head>

<title>

The <title> tag sets the title of the HTML document.

<title>My Website</title>

<body>

The <body> tag contains the content of the HTML document.

<body>
    <h1>Welcome to My Website</h1>
</body>

<h1> to <h6>

Heading tags <h1> to <h6> define headings, with <h1> being the highest level and <h6> the lowest.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

<p>

The <p> tag defines a paragraph.

<p>This is a paragraph.</p>

<a>

The <a> tag defines a hyperlink.

<a href="https://example.com">Visit Example</a>

<img>

The <img> tag embeds an image.

<img src="image.jpg" alt="Description">

<div>

The <div> tag defines a division or section in an HTML document.

<div>
    <p>This is a div section.</p>
</div>

<span>

The <span> tag defines a section in a document.

<span>This is a span section.</span>

<ul> and <ol>

The <ul> tag defines an unordered list, and the <ol> tag defines an ordered list.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

<li>

The <li> tag defines a list item.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<table>

The <table> tag defines a table.

<table border="1">
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
    </tr>
</table>

<tr>

The <tr> tag defines a table row.

<table border="1">
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
    </tr>
</table>

<td>

The <td> tag defines a table cell.

<table border="1">
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
    </tr>
</table>
