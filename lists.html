HTML Lists

HTML lists allow you to group a set of related items in lists. There are three types of lists in HTML: unordered lists, ordered lists, and definition lists.

Unordered Lists

An unordered list is a collection of items where the order does not matter. It is created using the <ul> and <li> tags.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Ordered Lists

An ordered list is a collection of items where the order matters. It is created using the <ol> and <li> tags.

<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

Definition Lists

A definition list is a collection of terms and their definitions. It is created using the <dl>, <dt> (definition term), and <dd> (definition description) tags.

<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
</dl>

Nested Lists

Lists can be nested inside other lists to create a hierarchical structure.

<ul>
    <li>Item 1
        <ul>
            <li>Subitem 1</li>
            <li>Subitem 2</li>
        </ul>
    </li>
    <li>Item 2</li>
</ul>
