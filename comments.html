HTML Comments and Notes

Comments are used in HTML to leave notes for developers and to temporarily disable parts of the code. Comments are not displayed in the browser.

HTML Comments

To add a comment in HTML, enclose the comment text within <!-- --> tags:

<!-- This is a comment -->

Comments can also be used to temporarily disable code:

<!-- <p>This paragraph will not be displayed.</p> -->

Important Notes

  • HTML comments are not displayed in the browser.
  • Comments are useful for leaving notes and explanations for developers.
  • Comments can help in debugging by isolating parts of the code.
  • Excessive comments can make the code cluttered. Use comments wisely.

Example

Here's an example of using comments in HTML:

<!DOCTYPE html>
<html>
    <head>
        <title>Comment Example</title>
    </head>
    <body>
        <h1>Hello World!</h1>
        <!-- This is a comment -->
        <p>This is a paragraph.</p>
        <!-- <p>This paragraph is commented out and will not be displayed.</p> -->
    </body>
</html>
